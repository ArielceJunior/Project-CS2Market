<h1 align="center">CS2Market</h1>
<h4 align='center'>Um projeto feito na linguagem C# e com conexão a um Banco de Dados</h2>
### Linguagens Utilizadas:
* C#
* SQL

### Ferramentas Utilizadas:
* Microsoft Visual Studio
* MySql Workbench
* HeidiSQL
* Xampp

### Funções do Software:
* Inserir dados;
* Listar dados;
* Alterar dados;
* Remover dados;
